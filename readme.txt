BLAZE WooCommerce Extension to sync your BLAZE products to your WooCommerce store!

== Description ==

This plugin allows you to integrate your BLAZE store with WooCommerce store. It helps to sync all the products added in BLAZE to the WooCommerce store
and manage all the orders from WooCommerce only. NOTE: THIS PLUGIN IS NO LONGER SUPPORTED OR DEVELOPED AND WILL BE REMOVED FROM THE DIRECTORY.

== Installation ==

1. Upload `blaze-retail-woocommerce` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the BLAZE API key on the WooCommerce general settings screen.
4. Done!
